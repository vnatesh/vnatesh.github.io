Network-wide Programming Model

For my master’s thesis, I have been working on a programming model, inspired by stream processing, that takes a high-level specification of a network program and compiles it into runnable code (in P4 or C) that can be launched on the programmable devices of network [2]. In this model, a network-wide program consists of a directed acyclic graph (DAG) where each node represents some high-level code snippet to be executed on packets arriving at that node and edges represent the order in which to execute the nodes. This thesis endeavors to make networks considerably easier to program..

Packet Stream-SQL

During the summer of 2018, I conducted an independent study to accelerate stream processing performance to terabit per second speeds using a virtual programmable switch. In this system, a packet contains a tuple payload with several named fields and corresponding values. Packets are continuously injected into a Mininet emulator where the virtual router filters packets based on a user query and forwards the results to a virtual host. I successfully implemented several SQL-like streaming primitives including filter (WHERE), project (SELECT), join (INNER JOIN), and aggregate operators (GROUPBY). To the right is a demo of a simple JOIN operation between two streams. Host 1 sends a stream of ad clicks while Host 2 sends a stream of ad impressions. The two streams are efficiently joined by the ad identifier at the virtual switch and the query result is sent to Host 3. SELECT adId, impression_time, click_time FROM impressions i INNER JOIN clicks c ON i.adId = c.adId

p-adic numbers and the Hasse-Minkowski Theorem

Here is a paper I wrote in 2014 on p-adic numbers as part of an honors Number Theory course and the Distinguished Major Program. I delivered a lecture on this topic. Given a polynomial f ∈ Z[x1, x2, . . . , xn], determing the existence of integral roots is a standard topic covered in an introductory number theory course. In this paper, we will extend our considerations to roots in Q. Using p-adic numbers, the existence of rational solutions can be readily acertained under select conditions, especially for quadratic forms

RSA public key encryption

This paper is a survey of RSA encryption I wrote during my second year (2012). The RSA public key cryptosystem is a method of encryption which was made public in 1977. Named after its inventors, Ron Rivest, Adi Shamir, and Leonard Adleman, the method relies on the fact that large integers, on the order of hundreds of digits, require an extraordinary amount of time for a computer to factor. This paper explores the general procedure for RSA encryption, as well as a look into tests of primality

Published Papers

Unique minimal forcing sets and forced representation of integers by quadratic forms

For S, W ⊆ N, we say S forces W, denoted S |-> |_| W, if any positive definite quadratic form with integer matrix which represents every element of S over Z also represents every element of W over Z. In the context of a superset S*, S is referred to as a unique minimal forcing set of W if for any S0 ⊆ S* we have that S0 |-> |_| W if and only if S ⊆ S0. In 2000, Manjul Bhargava used his own novel method of "escalators" to prove the unique minimal forcing set of N is T = {1, 2, 3, 5, 6, 7, 10, 14, 15}, which was a refinement of the celebrated Conway-Schneeberger Fifteen Theorem. The author uses Bhargava’s theory of escalators to establish several infinite familes of positive integers, interpreted as singletons in N, without unique minimal forcing sets in T. In this paper, these concepts are explored in greater detail.

Improving the error-correcting code used in 3-G communication

In 2011, Samsung Electronics Co. filed a complaint against Apple Inc. for alleged infringement of patents described in US 7706348, which details several embodiments of a TFCI (Transport Format Combination Indicator) encoder for mobile communication systems. One of the primary embodiments in question was a [30,10,10] non-cyclic code which was implemented in many devices communicating on the 3-G network, including several Apple products. However, the derivation of the basis for this code is left rather vague in the patent documentation. In this paper, the explicit construction of a [30, 10, 10] cyclic code is detailed using methods described by F.J. MacWilliams and N.J.A. Sloane in their well-known text, "The Theory of Error-Correcting Codes." We also give a construction of an optimal [30,10,11] non-cyclic code, which is distinct from the conventional and well-known construction involving manipulations of an extended BCH code.
